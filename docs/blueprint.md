# **App Name**: CreatorOS

## Core Features:

- Unified Dashboard: Displays key metrics such as upcoming content, revenue, and task lists in a customizable view.
- AI Content Studio: Uses multiple AI models to generate content with personalized brand voices and deep content repurposing tools, guided by SEO and suggested visuals; it offers a blog post writing tool featuring reasoning to decide whether to incorporate given SEO suggestions. This feature uses a tool.
- AI Content Calendar: Suggests content topics, posting schedules, and content types based on niche, audience analysis, and trending topics.
- Social Media Management: Links social media accounts to manage DMs, comments, and mentions in one inbox.
- Social Post Generation: Creates unique social post versions from a single idea, tailored for different platforms.
- Service Booking System: Creates services with descriptions, pricing, and integrates with personal calendar for booking.
- Client Portal: A secure area for clients to view their project status, booking history, access shared files, and communicate.

## Style Guidelines:

- Primary color: Saturated violet (#A052DE) for a creative and sophisticated feel.
- Background color: Light gray (#F0F0F0) to provide a clean and modern backdrop.
- Accent color: Yellow-orange (#FFB347) to highlight key elements and calls to action.
- Body font: 'Inter', a sans-serif font known for its legibility and clean design, suitable for body text and headlines.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist, outline-style icons to represent different features and categories.
- Incorporate subtle animations on content loading and transitions to enhance user experience.
